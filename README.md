# Projects-in-Python
Reposítório de projetos em python

#1 COOH PIAH
Aplicação que detecta casos de plágio em arquivos de texto.
O programa foi desenvolvido como trabalho de conclusão para o curso de introdução a ciência da computação com python.


#2 Jogo do Nim
O jogo consiste em uma estrutura lógica onde o jogador(ou máquina) a retirar a última peça ganha. O programa apresenta dois modos, campeonato e jogo rápido. Além de ser possível jogar conta outro jogador ou contra a máquina.

Aplicação desenvolvida como pré-requisito da primeira parte do curso de introdução a ciência da computação com python.
