#2 Jogo do Nim
O jogo consiste em uma estrutura lógica onde o jogador(ou máquina) ao retirar a última peça ganha. O programa apresenta dois modos, campeonato (onde o competidor contra máquina ou ou outro jogador disputam uma melhor de três) e jogo rápido(onde o jogador compete contra outro jogador ou máquina em uma única partida). A estrutura do jogo é feita de forma que o a máquina utilize a melhor combinação para retirar a últma peça do tabuleiro.

Aplicação desenvolvida como pré-requisito da primeira parte do curso de introdução a ciência da computação com python.
